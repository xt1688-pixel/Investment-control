# 投资控制看板

## 🚀 启动方式（推荐顺序）

### 方式一：macOS 应用（最方便）⭐

**双击 `投资看板.app` 文件**

- 可以拖到 Dock 或桌面，像普通应用一样使用
- 双击即可启动服务器并打开浏览器
- 最接近"一键启动"的体验

### 方式二：浏览器收藏夹（适合日常使用）⭐

1. **打开 `投资看板.html` 文件**（双击即可在浏览器中打开）
2. **将页面添加到浏览器收藏夹**：
   - Chrome/Edge: 按 `Cmd+D` 或点击地址栏的⭐图标
   - Safari: 按 `Cmd+D` 或点击"分享"按钮 → "添加书签"
3. **以后使用**：
   - 点击收藏夹中的"投资看板"
   - 如果服务器未运行，点击"启动并打开应用"按钮
   - 如果服务器已运行，会自动跳转到应用页面

### 方式三：启动脚本

**双击 `启动程序.command` 文件**，或在终端执行：

```bash
./启动程序.command
```

或

```bash
./start.sh
```

## 其他启动方式

### 方法一：激活虚拟环境后运行

1. **激活虚拟环境**：
```bash
source venv/bin/activate
```

2. **运行 Streamlit 应用**：
```bash
streamlit run app.py
```

### 方法二：直接使用虚拟环境中的 Python

```bash
./venv/bin/streamlit run app.py
```

---

## 数据自动保存

- ✅ 所有编辑的数据会自动保存到 `data` 文件夹
- ✅ 下次打开程序时会自动加载之前保存的数据
- ✅ 可以随时关闭浏览器，数据不会丢失

## 📱 手机访问

### 方法一：添加到主屏幕（推荐）
1. 在手机浏览器中打开投资看板
2. **iPhone**: 分享 → 添加到主屏幕
3. **Android**: 菜单 → 添加到主屏幕
4. 完成后像原生APP一样使用

### 方法二：局域网访问
1. 确保手机和电脑在同一WiFi
2. 运行程序后查看终端中的"Network URL"
3. 在手机浏览器输入该地址

详细说明请查看：`手机访问说明.md`

## 注意事项

- 程序会在浏览器中自动打开，默认地址：http://localhost:8501
- 如需停止程序，在终端按 `Ctrl+C`
- 如果端口被占用，Streamlit 会自动使用其他端口（8502、8503等）
- 手机访问时，确保手机和电脑在同一WiFi网络
